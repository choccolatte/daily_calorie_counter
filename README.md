# daily_calorie_counter
Using this online calorie counter, you can count whether you're in calorie-surplus (excess calories) or calorie-deficit (less calories).

Using it is fairly easy:

1. Enter the 'Calorie Budget' - the target calories to be consumed in a day.
2. Enter the breakfast, lunch, dinner, and snacks consumed during the day.
3. Enter the amount of exercise done during the day.
4. Click on 'Calculate Remaining Calories' to get details about the consumed calories.
